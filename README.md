# Simplex-and-Tableau-Simplex-Methods-for-Linear-Programs
Contains the Simplex method and Tableau Simplex method for solving linear programs in standard form.

Code written in Python

All linear programs are assumed to be in standard form with vairables $`\vec x \geq 0`$. Both methods also indicate when a linear program is infeasible or unbounded. 

Repository contains a simple example which finds a solution using each method under Simplex_Tests.py
